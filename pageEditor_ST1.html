Splash Page

Technical Description

  • <!DOCTYPE html>

    Defines the document type as HTML5 — tells the browser to render the page in standards-compliant mode.

  • <link rel="stylesheet"> and <link href="...">

    Used to link external resources: one for the Google Fonts (Poppins), and one for an external CSS file.

  • <body>

    Contains the visible content of the webpage such as headings, paragraphs, and interactive elements.

  • <div class="background">

    A full-screen container with a background image and overlay styling. Flexbox is used to center content.

  • <table>

    Used here to vertically align and structure the splash screen’s text and loader.

  • <p>

    Descriptive paragraph explaining the mission. Styling improves readability with controlled width and spacing.

  • <script> window.onload ... </script>

    JavaScript code that waits 3 seconds, adds a fade-out effect, and then redirects to home.html. Provides a timed splash screen.

  • <style> ... </style>

    Contains internal CSS to style the layout, text, background, and animations. Key features include a blurred background, flex centering, and an animated ring loader.

Accessibility

  • Imported google fonts for minimalistic and clear font styles.
  • A loader animation is included using CSS (@keyframes ring), and a "Directing..." message to indicate a loading animation.
  • Used high contrast between text and background for readability.

Volunteer Page

Technical Description

  • <div class="cards-container">

    A flex container that holds multiple volunteer program cards. Flexbox ensures responsiveness and wrapping of cards on smaller screens.

  • <div class="card">

    Represents a reusable UI block with an image, title, details, description, and link. Cards help structure and group related content.

  • <form> and <textarea>

    Used for user feedback. The textarea allows multiline input. Forms support interaction and data submission.

  • display: flex

    CSS property used for layout. Applied to containers like .cards-container and

  • object-fit: cover

    Ensures images fill their container without distortion by cropping excess parts. Maintains a clean layout for card images.

  • transition: transform 0.3s

    Adds smooth hover animations for cards, improving user interaction by subtly raising cards when hovered.

  • border-radius

    Rounds corners of elements like cards, buttons, and input fields to create a softer, modern design.

Accessibility

  • Added images to help screen readers to understand the visual content.
  • Used transtitions and animations to make the user expireince more interactive.
  • Used buttons for users to navigate to external webpages.

Content Page

Technical Description

  • <div class="contentpages">

    A wrapper that contains all the page content. Helps in grouping and layout management.

  • <div class="main2">

    Container for the main section of the page that holds the heading and action cards.

  • <div class="cards-wrapper">

    Holds all the individual action cards in a responsive layout.

  • <div class="action-card">

    Represents a single card that lists ways to take action against climate change.

  • .main2 { padding: 50px 20px; }

    Adds space around the main content to prevent it from touching the browser edges.

  • .cards-wrapper { display: flex; flex-wrap: wrap; gap: 20px; justify-content: center; }

    Uses Flexbox to layout cards in a responsive, multi-row format with equal spacing.

  • .action-card { background-color: rgba(255,255,255,0.4); backdrop-filter: blur(8px); border-radius: 30px; }

    Gives each card a glassmorphic, modern UI effect with rounded corners and semi-transparent background.

  • .action-card:hover { transform: translateY(-5px); }

    Adds an animation on hover to lift the card slightly, making it more interactive. Controls spacing between footer lines to make the text easier to read.

Accessibility

  • Used a card for each action, used points to avoid lengthy descriptions.
  • Used a backdrop blur filter to have a modern eyecatching style.
