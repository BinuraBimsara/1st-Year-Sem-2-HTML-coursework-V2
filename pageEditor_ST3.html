Feedback Page

Technical Description

  • The form layout is structured into two main sections using <fieldset> for semantic grouping.
  • All form fields are styled with consistent spacing, colors, and rounded corners for a clean user experience.
  • A background image is fixed to the full page using background-attachment: fixed; for a modern look.
  • The form includes a star-based rating system using styled <ul> and <li> tags.
  • Responsive behavior is managed using @media queries to ensure good display on smaller screens.

Accessibility

  • All inputs have associated <label> tags using for and id attributes.
  • Elements are keyboard-accessible and visually readable across devices.
  • Form uses appropriate semantic HTML5 tags like <textarea>, <select>, and <datalist>.

Team Page

Technical Description

  • Used Flexbox via .team-grid to align team cards responsively.
  • Each member is presented using <div class="team-member"> containing a circular image, name, and role.
  • On hover, transform: translateY is applied to animate the cards and reveal more details.
  • Card design is modern, clean, and mobile-friendly.

Accessibility

  • Images use descriptive alt text.
  • Hover animations enhance interactivity without breaking layout structure.
  • Text color and contrast meet WCAG standards.
